## 工作原理

### 传统转账 vs 混币器

**传统转账**：
```
地址 A → 地址 B
```
链上可以直接看到 A 转给了 B

**混币器转账**：
```
地址 A
  ↓
中间地址 1 → 中间地址 5 → 中间地址 20
中间地址 2 → 中间地址 10 → 中间地址 30
中间地址 3 → 中间地址 15 → 中间地址 40
  ↓
地址 B
```
链上很难追踪 A 到 B 的关系

## 核心功能

### 1. 多跳转账
- 资金不是直接从源地址到目标地址
- 而是通过 3-10 跳的中间地址
- 每跳之间有时间延迟

### 2. 交叉路径
- 生成多条独立路径
- 路径之间可能交叉
- 增加追踪难度

### 3. 自动收费
- 服务费自动转到指定地址
- 费率根据跳数计算：
  - 10 跳：1%
  - 50 跳：2%
  - 100 跳：3%
  - 500 跳：15%
  - 1000 跳：30%

### 4. Gas 费用
- 自动估算 Gas 费用
- 从总金额中扣除
- 用户承担所有费用

## 使用方式

### 方式 1: Python 脚本

```python
from mixer_engine import MixerEngine
from hd_wallet import HDWallet

# 初始化
mnemonic = "你的助记词"
wallet = HDWallet(mnemonic)
engine = MixerEngine('bsc_testnet')  # 或 'bsc'

# 获取源地址和目标地址
source = wallet.get_account(0)
target = wallet.get_account(10)

# 创建混币计划
plan = engine.create_mixing_plan(
    from_private_key=source['private_key'],
    to_address=target['address'],
    total_amount=0.1,  # 0.1 BNB
    num_hops=10,       # 10 跳
    mnemonic=mnemonic
)

# 执行混币
result = engine.execute_mixing(
    plan=plan,
    gas_level='standard',
    delay_range=(1, 3)  # 每跳延迟 1-3 秒
)

print(f"成功: {result['success_count']}")
print(f"目标地址收到: {result['total_collected']} BNB")
print(f"服务费: {result['service_fee']} BNB")
```

## 安全建议

### 1. 测试先行
- 先在测试网测试
- 确认流程无误后再用主网
- 小额测试

### 2. 助记词安全
- 妥善保管助记词
- 不要泄露给任何人
- 离线备份

### 3. 金额控制
- 不要一次转太大金额
- 分批转账更安全
- 留足 Gas 费用

### 4. 时间延迟
- 建议设置较长延迟（几分钟）
- 避免被识别为机器人
- 更像真实用户行为

### 5. 跳数选择
- 跳数越多，隐私性越好
- 但费用也越高
- 建议 10-100 跳

## 常见问题

### Q: 为什么有些交易失败？
A: 可能原因：
- 余额不足（小额地址无法支付 Gas）
- Nonce 冲突（已优化）
- 网络拥堵

### Q: 服务费地址是什么？
A: 0xE602348170Bc045c588bF1638b0EdC592F767250

### Q: 可以自定义服务费吗？
A: 当前版本不支持，费率固定

### Q: 支持哪些链？
A: 当前支持：
- BSC 主网
- BSC 测试网

即将支持：
- Ethereum
- Polygon
- Arbitrum

### Q: 如何查看交易记录？
A: 每笔交易都会返回 explorer_url，可以在区块链浏览器查看

### Q: 混币需要多长时间？
A: 取决于跳数和延迟设置：
- 10 跳，1-3 秒延迟：约 30-60 秒
- 100 跳，1-3 秒延迟：约 5-10 分钟
- 建议设置更长延迟以增加隐私性

## 技术细节

### 路径生成算法
- 随机选择中间地址
- 避免重复路径
- 支持循环路径

### Nonce 管理
- 自动获取当前 nonce
- 顺序递增避免冲突
- 支持并发转账

### 余额检查
- 每次转账前检查余额
- 预留 Gas 费用（0.0003 BNB）
- 自动跳过余额不足的地址

### 金额分配
- 随机分配到各路径
- 每条路径金额略有差异
- 增加隐私性
