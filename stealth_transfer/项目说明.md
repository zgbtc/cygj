# 隐私批量转账工具 - 项目说明

## 🎉 项目已完成！

这是一个功能完整的隐私批量转账工具，参考了 ct.app/stealth-transfer 的设计理念。

## 核心功能

### 1. 隐私转账 ⭐
- 从助记词自动生成 10-10000 个地址
- 将资金分散到这些地址，隐藏资金来源
- 支持随机分配和平均分配
- 所有地址可用助记词恢复

### 2. 普通批量转账
- 手动指定接收地址列表
- 批量发送到 10-10000 个地址
- 支持自定义每个地址的金额

### 3. 多种使用方式
- Web 界面（最友好）
- 命令行工具（交互式）
- Python API（编程集成）
- REST API（远程调用）

## 快速开始

### 1. 安装依赖
```bash
cd stealth_transfer
pip install -r requirements.txt
```

### 2. 启动服务
```bash
python app.py
```

### 3. 访问界面
- 隐私转账：http://localhost:5000/stealth.html
- 普通转账：http://localhost:5000/

## 使用场景

### 场景 1: 隐私保护
你有 100 BNB 想要转移，但不想让别人追踪到你的主地址：
1. 使用隐私转账功能
2. 生成 100 个地址
3. 随机分配金额
4. 保存助记词
5. 需要时用助记词恢复任意地址

### 场景 2: 批量发放
你需要给 500 个用户发放代币：
1. 准备地址列表和金额
2. 使用普通批量转账
3. 一次性完成所有转账

### 场景 3: 测试开发
你在开发 DApp，需要多个测试地址：
1. 使用 HD 钱包生成地址
2. 用隐私转账给这些地址充值
3. 用助记词在 MetaMask 中切换账户

## 技术特点

### 1. BIP44 标准
- 符合行业标准的 HD 钱包实现
- 与 MetaMask、Trust Wallet 等兼容
- 路径：m/44'/60'/0'/0/{index}

### 2. 安全性
- 私钥本地处理，不上传服务器
- 支持离线签名
- 助记词加密存储（可选）

### 3. 性能
- 生成 1000 个地址：~9 秒
- 批量转账：并发处理
- 自动 nonce 管理

### 4. 可扩展性
- 模块化设计
- 易于添加新链支持
- 支持自定义 RPC 节点

## API 端点

### 隐私转账
- `POST /api/stealth/generate` - 生成地址
- `POST /api/stealth/plan` - 创建计划
- `POST /api/stealth/execute` - 执行转账
- `POST /api/stealth/estimate` - 估算费用

### 普通转账
- `GET /api/chains` - 获取支持的链
- `POST /api/estimate` - 估算费用
- `POST /api/balance` - 查询余额
- `POST /api/transfer` - 批量转账
- `POST /api/validate` - 验证地址

## 扩展开发

### 添加新链
1. 在 `config.py` 中添加链配置
2. 如果是 EVM 兼容链，无需修改代码
3. 如果是非 EVM 链（如 Solana），需要实现对应的转账逻辑

### 添加 ERC20 支持
1. 在 `transfer_engine.py` 中添加 ERC20 转账方法
2. 更新 API 端点
3. 更新前端界面

### 添加更多分配策略
在 `stealth_engine.py` 的 `_distribute_amount` 方法中添加新策略。

- ⚠️ 使用硬件钱包

## 已知限制

1. **仅支持原生代币**：当前版本不支持 ERC20
2. **单线程处理**：转账是串行的，大量地址需要时间
3. **Gas 价格固定**：不支持动态 Gas 价格
4. **无交易确认**：不等待交易确认，仅发送
