我们已经成功集成了以下高级功能：

## 🐛 常见问题

### 1. Tor 连接失败

**问题**: `❌ Tor 连接失败`

**解决**:
```bash
# 检查 Tor 是否运行
netstat -an | grep 9050  # Linux/Mac
netstat -an | findstr 9050  # Windows

# 重启 Tor
sudo service tor restart  # Linux
brew services restart tor  # Mac
# Windows: 重新运行 tor.exe
```

### 2. 跨链桥接失败

**问题**: `❌ 跨链桥接失败`

**解决**:
- 检查源链余额是否足够（包括桥接费）
- 等待桥接确认（5-30分钟）
- 检查目标链 RPC 是否正常

### 3. 交易卡住

**问题**: 交易长时间 pending

**解决**:
- 检查 Gas 价格是否过低
- 检查网络是否拥堵
- 等待或取消重发（增加 Gas）

### 4. 余额不足

**问题**: `余额不足`

**解决**:
```
总金额 = 转账金额 + 服务费 + Gas费 + 跨链费

示例（100跳隐私模式）:
  转账: 0.1 BNB
  服务费: 0.03 BNB
  Gas: 0.021 BNB
  总需要: 0.151 BNB
```

## 🎓 技术细节

### 多路径混币算法

```python
# 阶段 1: 分散（20% 地址）
num_start = max(5, num_hops // 5)
amounts = random_split(total, num_start)
for i, amount in enumerate(amounts):
    transfer(source, start_addresses[i], amount)
    delay(5-30 min)

# 阶段 2: 随机交叉跳转
while hops < target_hops:
    from_addr = random.choice(active_addresses)
    to_addr = random.choice(all_addresses - from_addr)
    amount = balance(from_addr) - gas_reserve
    transfer(from_addr, to_addr, amount)
    delay(5-30 min)

# 阶段 3: 汇总
for addr in all_addresses:
    if balance(addr) > gas_reserve:
        transfer(addr, target, balance - gas_reserve)
        delay(5-30 min)
```
