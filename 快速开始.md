## 📝 使用步骤

### 1. 访问网站
```
https://cygj-crypto-tools.vercel.app
```

### 2. 选择模式
- 新手/测试 → **快速模式**
- 正常使用 → **隐私模式**
- 大额资金 → **极致隐私**

### 3. 填写参数
```
网络: BSC Testnet (测试) 或 BSC Mainnet (正式)
源地址私钥: 0x...
目标地址: 0x...
转账金额: 0.1 BNB
跳数: 100
```

### 4. 执行混币
点击"🚀 执行混币"按钮

### 5. 等待完成
- 快速模式: 30 分钟
- 隐私模式: 1-2 天
- 极致模式: 3-7 天

## 🔐 安装 Tor（隐私模式需要）

### Windows
1. 下载: https://www.torproject.org/download/tor/
2. 解压并运行 `tor.exe`
3. 默认端口: 127.0.0.1:9050

### Linux
```bash
sudo apt-get install tor
sudo service tor start
```

### Mac
```bash
brew install tor
brew services start tor
```

### 验证 Tor
```bash
curl --socks5 127.0.0.1:9050 https://check.torproject.org
```

## 🐛 常见问题

### Q: Tor 连接失败？
```bash
# 检查 Tor 是否运行
netstat -an | grep 9050

# 重启 Tor
sudo service tor restart
```

### Q: 余额不足？
```
确保余额 >= 转账金额 + 总费用

示例: 转 0.1 BNB，需要至少 0.151 BNB
```

### Q: 交易卡住？
```
- 检查 Gas 价格
- 检查网络状态
- 等待或取消重发
```
