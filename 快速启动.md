## 🎯 测试流程

### 在测试网测试（推荐）

1. **准备测试币**
   - 访问 BSC Testnet Faucet
   - 获取一些 tBNB 测试币

2. **使用工具**
   - 访问 http://localhost:5000/stealth.html
   - 选择 "BSC Testnet"
   - 输入你的私钥或助记词
   - 设置参数：
     - 总金额: 0.1 tBNB
     - 目标地址: 你的另一个地址
     - 跳数: 10 或 100
   - 点击 "执行混币"

3. **查看结果**
   - 等待交易完成
   - 在 BSCScan Testnet 查看交易
   - 确认目标地址收到资金

### 测试助记词
```
decline cluster album give brief scrap apart onion dust donor figure primary
```

## 🛠️ 管理服务器

### 查看服务器状态
两个服务器都在后台运行，你可以：
- 查看终端输出
- 检查日志
- 监控性能

### 停止服务器
如果需要停止服务器：
1. 在 Kiro 中找到正在运行的进程
2. 停止对应的进程
3. 或者在终端按 `Ctrl+C`

### 重启服务器
如果需要重启：

**重启前端**:
```bash
cd crypto-tools-hub
npm run dev
```

**重启后端**:
```bash
cd stealth_transfer
python app.py
```
