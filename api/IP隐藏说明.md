# IP隐藏功能说明

## 实现方案

### 快速模式
- **IP隐藏**: 建议用户自行使用VPN
- **原因**: 快速模式注重速度，不启用代理池
- **提示**: 前端会显示"⚠️ IP隐藏: 建议使用VPN"

### 极致隐私模式
- **IP隐藏**: 自动启用免费代理池
- **原理**: 
  1. 从多个公开源抓取免费代理
  2. 并发验证代理可用性
  3. 每次交易随机选择代理
  4. 自动轮换，类似Tor效果
- **提示**: 前端会显示"🔒 IP隐藏: 已启用（代理池）"

## 代理池特性

### 代理来源
1. proxy-list.download
2. free-proxy-list.net
3. pubproxy.com
4. geonode.com

### 自动管理
- ✅ 自动抓取和验证
- ✅ 按响应速度排序
- ✅ 自动移除失效代理
- ✅ 每小时自动刷新
- ✅ 并发测试（20线程）

### 使用方式

```python
# 启用代理池
mixer = AdvancedMixerEngine(chain='bsc_testnet', mode='ultimate')

# 代理池会自动：
# 1. 抓取免费代理
# 2. 验证可用性
# 3. 每次交易随机选择
# 4. 失败自动切换
```

## 注意事项

1. **免费代理稳定性**: 免费代理可能不稳定，系统会自动重试
2. **速度影响**: 使用代理会增加延迟（通常1-3秒）
3. **自动降级**: 如果代理池为空，自动使用直连
4. **隐私保护**: 每次交易使用不同IP，难以追踪

## 优势

✅ **完全免费**: 不需要付费Tor或VPN服务
✅ **自动化**: 无需手动配置
✅ **高可用**: 多源抓取，自动验证
✅ **真实隐藏**: 每次交易不同IP
✅ **无需本地**: 服务器端自动管理
