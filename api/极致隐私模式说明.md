# 极致隐私模式说明

## 核心特性

### 1. 真正无法追溯的跨链混币

**关键技术：临时地址隔离**

传统跨链混币的问题：
```
源地址A (BSC) → 混币 → 地址A (Polygon) → 混币 → 地址A (Arbitrum)
❌ 跨链桥可以看到：地址A在不同链之间转账，可以追溯！
```

我们的解决方案：
```
源地址A (BSC)
  ↓ 混币25跳
临时地址B (BSC) ← 混币最后一跳到这里
  ↓ 跨链桥
临时地址C (Polygon) ← 跨链到达新地址
  ↓ 混币25跳  
临时地址D (Polygon)
  ↓ 跨链桥
临时地址E (Arbitrum) ← 又是新地址
  ↓ 混币25跳
临时地址F (Arbitrum)
  ↓ 跨链桥
目标地址G (BSC) ← 最终到达用户指定地址

✅ 跨链桥只能看到：
   - B → C（无法关联到A）
   - D → E（无法关联到B、C）
   - F → G（无法关联到D、E、F）
```

**实现细节：**
1. 使用HD钱包为每次跨链生成新的临时地址
2. 每次跨链前，先混币到临时地址
3. 用临时地址的私钥执行跨链
4. 到新链后，从临时地址继续混币
5. 所有临时地址都是一次性使用，用完即弃

### 2. 跨链路径

**默认路径：** BSC → Polygon → Arbitrum → BSC

- 支持68+条链
- 每条链上执行25跳混币
- 总共100跳，完全打乱资金路径

## 使用方法

```python
from advanced_mixer_engine import AdvancedMixerEngine

# 创建极致隐私混币器
mixer = AdvancedMixerEngine(chain='bsc', mode='ultimate')

# 创建混币计划
plan = mixer.create_mixing_plan(
    from_private_key='your_private_key',
    to_address='target_address',
    total_amount=1.0,  # 1 BNB
    num_hops=100,
    mnemonic='your_mnemonic'  # 用于生成临时地址
)

# 执行混币
result = mixer.execute_mixing(plan)
```
