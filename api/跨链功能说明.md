# 跨链功能说明

## 当前实现状态

### ✅ 已实现（简化版）
- 多链连接管理
- 跨链路径规划
- 基础跨链转账框架

### ⚠️ 简化说明
当前实现是**简化版本**，使用直接转账模拟跨链效果。

**原因：**
1. 真正的Stargate跨链需要大量Gas费测试
2. 需要USDC/USDT等稳定币（不是原生币）
3. 测试网的跨链桥可能不稳定
4. 实现复杂度高，需要深入测试

### 🎯 实际效果
虽然是简化版，但在**极致隐私模式**下：
1. 在BSC上混币（多跳）
2. 提示"建议跨链到Polygon继续混币"
3. 用户可手动使用任何跨链桥
4. 在Polygon上继续混币
5. 最终达到多链混淆效果

### 核心代码示例

```python
# Stargate Router合约调用
def bridge_with_stargate(
    from_chain: str,
    to_chain: str,
    amount: float,
    private_key: str
):
    # 1. 连接Stargate Router
    router_address = STARGATE_ROUTERS[from_chain]
    router = w3.eth.contract(address=router_address, abi=STARGATE_ROUTER_ABI)
    
    # 2. 准备参数
    dst_chain_id = LAYERZERO_CHAIN_IDS[to_chain]
    src_pool_id = POOL_IDS[from_chain]['usdc']
    dst_pool_id = POOL_IDS[to_chain]['usdc']
    
    # 3. 估算费用
    (native_fee, zro_fee) = router.functions.quoteLayerZeroFee(
        dst_chain_id,
        1,  # TYPE_SWAP_REMOTE
        to_address_bytes,
        b'',
        (0, 0, b'')
    ).call()
    
    # 4. 执行跨链
    tx = router.functions.swap(
        dst_chain_id,
        src_pool_id,
        dst_pool_id,
        refund_address,
        amount_wei,
        min_amount_wei,
        (0, 0, b''),  # lzTxParams
        to_address_bytes,
        b''  # payload
    ).build_transaction({
        'from': from_address,
        'value': native_fee,
        'gas': 500000,
        'gasPrice': gas_price,
        'nonce': nonce
    })
    
    # 5. 签名发送
    signed_tx = w3.eth.account.sign_transaction(tx, private_key)
    tx_hash = w3.eth.send_raw_transaction(signed_tx.rawTransaction)
```

## 费用说明

### Stargate跨链费用
1. **LayerZero消息费**：~$0.1-0.5（根据链而定）
2. **目标链Gas费**：~$0.05-0.2
3. **Stargate协议费**：0.06%
4. **滑点**：0.1-1%

### 总费用估算
- 小额（<$100）：~$0.5-1
- 中额（$100-1000）：~$1-3
- 大额（>$1000）：~$3-10

## 使用建议

### 当前版本（简化版）
```python
# 极致隐私模式会提示跨链
mixer = AdvancedMixerEngine(chain='bsc', mode='ultimate')

# 执行混币
result = mixer.execute_mixing(plan)

# 用户看到提示：
# "💡 建议跨链到Polygon继续混币以增强隐私"
```

### 手动跨链方案
1. 在BSC上混币完成
2. 使用任何跨链桥（Stargate/Multichain/cBridge）
3. 桥接到Polygon
4. 在Polygon上继续混币
5. 桥回BSC或直接发送到目标地址

### 推荐跨链桥
- **Stargate Finance**: 最安全，基于LayerZero
- **Multichain**: 支持链最多
- **cBridge**: Celer Network，速度快
