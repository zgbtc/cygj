## 🔧 需要注意的问题

### 1. Polygon RPC 连接
```
⚠️ Polygon RPC 偶尔连接失败
解决方案: 可更换为其他 Polygon RPC
- https://polygon-rpc.com
- https://rpc-mainnet.matic.network
- https://matic-mainnet.chainstacklabs.com
```

### 2. Tor 依赖
```
⚠️ 隐私模式和极致模式需要 Tor
用户需要:
1. 安装 Tor
2. 启动 Tor 服务（127.0.0.1:9050）
3. 验证连接

如果没有 Tor，会回退到直接连接（隐私性降低）
```

### 3. 跨链桥接（极致模式）
```
⚠️ 跨链桥接需要额外时间
- 每次桥接: 5-30 分钟
- 3次跨链: 15-90 分钟额外时间
- 需要足够的 Gas 费

建议: 先在测试网测试跨链流程
```
